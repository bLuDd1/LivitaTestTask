# iOS Test Task

A native iOS application implemented in **SwiftUI** based on the provided Figma design.

## How to Run

1. **Clone or download** this repository/archive.
2. Open **Xcode** (v26.3 or newer recommended).
3. Select `File` -> `Open...` and choose the project directory or `.xcodeproj` file.
4. Select an iPhone simulator target that supports iOS 18.0.
5. Press Cmd + R to build and run the application.
